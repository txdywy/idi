# Public Feature Map Research

Source: public product page provided by the user, summarized at category level only.

## Broad feature categories
- Menu bar monitoring with configurable compact readouts.
- CPU/GPU activity, frequency, thermal, and top process attribution.
- Memory pressure, swap, compression, and top memory apps.
- Disk capacity, activity, health, and app-level activity.
- Network throughput, interfaces, addresses, connectivity, and app-level usage.
- Battery and power state for Mac and connected devices.
- Sensors, fans, voltages, current, power, and temperature thresholds.
- Weather, forecasts, date/time, clocks, calendars, and astronomical context.
- Rules and alerts for threshold-driven notifications.
- Customization for layout density, colors, order, and combined views.

## idi interpretation
idi will treat these as category expectations for a complete system monitor, but it will use its own information architecture, typography, visual system, interaction model, and implementation.
